# 🚀 AI Prompt Reconstructor & Enhancer

**AI Prompt Reconstructor** is an advanced AI-powered tool designed to analyze YouTube content and reverse-engineer the prompt engineering behind it. It extracts transcripts, identifies core themes, and generates high-quality, professional-grade prompts that you can use with any LLM like ChatGPT, Claude, or Gemini.

🌟 **Features**
*   🎥 **YouTube Transcript Extraction**: Supports standard URLs, shortened links (`youtu.be`), and YouTube Shorts.
*   🤖 **AI Prompt Reconstruction**: Intelligently analyzes video content to discover its underlying "prompt logic."
*   ⚡ **Power Prompt Enhancement**: Automatically generates supercharged versions of prompts for maximum output quality.
*   🔑 **Flexible API Configuration**: Persistent API key entry directly through the web UI with real-time server-side validation.
*   🔄 **Gemini REST Fallback**: Specialized architecture ensures full Gemini AI functionality even on experimental Python environments (like Python 3.14).
*   🎨 **Premium UI/UX**: A sleek, modern glassmorphic dashboard with smooth micro-animations and a responsive mobile-first design.

🛠️ **Technology Stack**
*   **Frontend**: Vanilla HTML5, CSS3 (Modern Glassmorphism), JavaScript (ES6+)
*   **Backend**: Flask (Python)
*   **AI Engine**: Google Gemini API (supporting both Official SDK and REST Fallback)
*   **YouTube API**: `youtube-transcript-api`
*   **Environment**: `python-dotenv`
*   **Icons & Fonts**: FontAwesome, Google Fonts (Inter, JetBrains Mono)

🚀 **Getting Started**
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/subhankashif4004-droid/AI-Prompt-Reconstructor.git
    cd AI-Prompt-Reconstructor
    ```
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the Application**:
    ```bash
    python app.py
    ```
4.  **Set Up API Key**: 
    Open **[http://localhost:5000](http://localhost:5000)** and enter your Gemini API key in the header to unlock the Pro Engine.
